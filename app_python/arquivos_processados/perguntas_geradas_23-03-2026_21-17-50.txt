CATEGORIA: Nomenclatura de Objetos
========================================
PERGUNTA: Estou desenvolvendo um procedure para atualizar dados de veículos roubados. Qual seria o nome correto para este procedimento, considerando a estrutura do DETRAN-PE?
========================================
RESPOSTA:
O nome deve seguir a estrutura NomeObjetivoOperacao. No exemplo Operacao..VeiculoRoubadoAtualizarS.scp, a operação é 'Atualizar' e a sigla é 'S' para Selecionar/Alterar. Portanto, o nome correto é VeiculoRoubadoAtualizarS.scp.

CATEGORIA: Boas Práticas
========================================
PERGUNTA: Posso usar o prefixo 'tmp' para um procedure temporário no DETRAN-PE?
========================================
RESPOSTA:
Não. De acordo com as boas práticas, o prefixo 'tmp' é exclusivo para tabelas temporárias. Procedures devem utilizar nomes sem este prefixo. Exemplo correto: dbfisc01..NomeProcedure.scp sem o prefixo 'tmp'.

CATEGORIA: Operações Básicas
========================================
PERGUNTA: Um colega está desenvolvendo um procedure para excluir dados de multas. Ele nomeou como 'ExcluirMultasI'. Isso está correto?
========================================
RESPOSTA:
Não. A operação de exclusão deve terminar com 'E'. Exemplo: dbfisc01..ExcluirMultasE.scp. A sigla 'E' é a única permitida para exclusão no padrão DETRAN-PE.

CATEGORIA: Nomenclatura de Objetos
========================================
PERGUNTA: Preciso criar uma procedure para consultar processos administrativos. Qual a estrutura correta para nomeá-la?
========================================
RESPOSTA:
A estrutura é NomeObjetivoOperacao. Exemplo: ControleProcessoAdmConsultarS.scp, onde 'Controle' é o substantivo, 'ProcessoAdm' define o objetivo e 'S' indica a operação de Selecionar.

CATEGORIA: Stored Procedures
========================================
PERGUNTA: Vi um código com procedure nomeado como 'RelatorioEstatisticasR'. Isso segue o padrão DETRAN-PE?
========================================
RESPOSTA:
Sim, está correto. O padrão permite a operação 'R' para Relatório. O exemplo ObterEstatisticasR.scp segue a mesma regra. A estrutura NomeObjetivoOperacaoR.scp é válida.
