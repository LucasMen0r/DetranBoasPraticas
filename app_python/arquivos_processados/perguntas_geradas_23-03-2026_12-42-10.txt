CATEGORIA: Nomenclatura de Objetos
========================================
PERGUNTA: Qual é o nome correto para uma procedure que atualiza dados de veículos?
========================================
RESPOSTA:
Procedimento..VeiculoAtualizarI.scp. De acordo com o padrão, o objetivo deve ser 'Veiculo' (resposta para 'O que inserir?') e a operação 'Atualizar' deve ser representada por 'I'.

CATEGORIA: Boas Práticas
========================================
PERGUNTA: Posso usar tmp_veiculo_sefaz para uma procedure temporária?
========================================
RESPOSTA:
Não. O prefixo 'tmp' é exclusivo para tabelas temporárias, conforme exemplo 'tmpParcelaDebitoSefaz' (Art. 5.2.3). Procedures devem usar extensão .scp.

CATEGORIA: Stored Procedures
========================================
PERGUNTA: Como devo nomear uma procedure que consulta histórico de multas?
========================================
RESPOSTA:
ConsultaHistoricoMultasS.scp. O objetivo 'Consulta' responde 'O que selecionar?' e a operação 'Selecionar' é representada por 'S'.

CATEGORIA: Integridade Referencial
========================================
PERGUNTA: O nome 'ValidarEspecialidadeMotoristaA' está correto?
========================================
RESPOSTA:
Sim. O objetivo 'ValidarEspecialidade' responde 'O que verificar?' e a operação 'Validar' corresponde à operação 'Alterar' (A), conforme regra de mapeamento operacional.

CATEGORIA: Nomenclatura de Objetos
========================================
PERGUNTA: Qual é o padrão correto para nomear uma tabela de histórico de multas?
========================================
RESPOSTA:
tbHistoricoMultas. O prefixo 'tb' indica tabela física, sigla deve ser usada (Art. 3.1.2), e o nome deve ser conciso (contrario a 'tbHistoricoMultasDetranPE' que é excessivo).
