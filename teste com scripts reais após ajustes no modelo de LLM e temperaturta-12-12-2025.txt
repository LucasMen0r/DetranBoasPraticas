PERGUNTA: Qual a forma correta de nomear uma chave estrangeira, uma chave primária e uma tabela temporária?
========================================
RESPOSTA:
The correct naming conventions for database elements according to the specified rules are as follows:

1. **Foreign Key**: Name should start with "fk" followed by the related table name. Example: `fkModeloVeiculo`.

2. **Primary Key**: Use "pk" followed by the table name. Example: `pkUsuario`.

3. **Temporary Table**: Prefix with "tmp" or "temp" to indicate it's temporary. Example: `tmpDados`.

These names adhere to the rules of avoiding reserved words, using appropriate prefixes, and keeping within 30 characters.
